#  Descripción de los componentes de tres páginas de Airbnb
Implementación de tres vistas de la experiencia de Airbnb en Next.js usando componentes de React y Next.js: la página de inicio (Home), la página de catálogo (resultados de búsqueda) y la vista de detalle de una habitación.

# Condiciones generales:
La implementación debe ser mobile-first. Se diseña primero para un viewport de 375px; se adapta al escritorio a partir de 768px.

# stack:
- Next.js 16
- React 19
- typescript 5
- tailwindCSS 4
- ESLint 9

# Descrición de cada página del proyecto:
A continuación se describe qué contiene cada página, cuáles son los componentes principales, quién es el usuario y qué intenta hacer. 
A partir de la página de airbnb se genera una especificación de los componentes de cada vista (una responsabilidad por componente) para que sirva de guia a esta implementación.
La navegación entre las tres páginas funciona sin recargar el navegador.

## Home (Página de Inicio):
Esta página muestra de una vez toda la oferta que tiene activa el sitio.


## Catálogo (Resultados de la busqueda de usuario)
El usuario llena las condiciones de filtraje que requiere y con eso le aparecen las opciones que cumplen su solicitud


## Detalle de una habitacion o propiedad
Esta vista muestra el detalle de una habitación una vez que el usuario da click en alguna de las opciones del catálogo.
