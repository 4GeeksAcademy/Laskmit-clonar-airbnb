# Laskmit-clonar-airbnb
Proyecto para clonar la pagina de Airbnb

# Resumen del proyecto
El proyecto consiste en clonar la arquitectura de componentes de Airbnb. 
La implementación debe ser mobile-first. Diseña primero para un viewport de 375px; adapta al escritorio a partir de 768px.

# Contexto del proyecto:

En este proyecto se definen diferentes contextos o descripciones del diseño efectuado.  Estos son:

## Archivo context.md:
En la raíz del repositorio, se describe la interfaz con sus componentes.  Son tres páginas las que se van a clonar: 1) La página principal, 2) El catálogo donde aparecen las opciones que corresponden a la busqueda que hace el usuario y 3) El detalle que aparece cuando se le da click para ver una propiedad en particular.  Para cada una de las páginas se definen las especificaciones funcionales y los componentes claves a desarrollar.

## Archivo design.md:
Describe el estilo de diseño definido.  Este diseño fué generado con google.stitch dándole como parámetro, similitud con la pagina de Airbnb.  Además con esta IA se hicieron las modificaciones al archivo de contexto necesarias según el enunciado del proyecto.

## Archivo components.md:
Es un detalle de los componentes principales del proyecto.  Aun cuando se habian descrito en el context.md, se detallaron aqui junto con el stack tecnológico requerido.

## Archivo Arquitectura.md:
Este documento describe la estructura de páginas, layout y componentes principales del proyecto, explicando para qué sirve y cómo funciona cada elemento clave.  Se incluye un diagrama tipo MERMAID para visualizar los elementos y componentes que usa cada página, incluyendo los hooks que se utilizaron.  Es un documento para comprender la estructura de un proyecto REACT con Next y le fué solicitado a la IA con fines didacticos.
