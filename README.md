# Laskmit-clonar-airbnb
Proyecto para clonar la pagina de Airbnb

# Resumen del proyecto
El proyecto consiste en clonar la arquitectura de componentes de Airbnb. 
La implementación debe ser mobile-first. Diseña primero para un viewport de 375px; adapta al escritorio a partir de 768px.

# Contexto del proyecto:

En el archivo context.md en la raíz del repositorio, se describe la interfaz con sus componentes.  Son tres páginas las que se van a clonar: 1) La página principal, 2) El catálogo donde aparecen las opciones que corresponden a la busqueda que hace el usuario y 3) El detalle que aparece cuando se le da click para ver una propiedad en particular.
